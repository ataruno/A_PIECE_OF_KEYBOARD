# A PIECE OF KEYBOARDのビルドガイド

## キット内容
以下の内容がキットに含まれます。  
| ケース関係               | 個数    |
| :----------------------- | ------- |
| キーキャップ＆上部パーツ | 1セット |
| ミドルプレート           | 1個     |
| ボトムケース             | 1個     |
| ネジ長                   | 3個     |
| ネジ短                   | 1個     |

| 部品関係         | 個数        |
| :--------------- | :---------- |
| 基板             | 1枚         |
| キースイッチ     | 9個         |
| ダイオード       | 9個+予備1個 |
| ホットスワップ   | 7個         |
| DIPスイッチ      | 1個         |
| リセットスイッチ | 1個         |
| 20pinコネクタ    | 1セット     |

### スイッチについて
Kailh Wizard Switch (Tactile Switch)
注：ビルドガイドではKailh Ghost Switchが映っていますが、<br>キットにはWizard Switchが含まれます。
